# Accept.Blue v2 Payment Gateway for WooCommerce

A powerful WooCommerce payment gateway plugin supporting Accept.Blue API v2 features, including:
- PCI-Compliant Payments and Hosted Tokenization
- Google Pay & 3D Secure Checkout
- Subscription and Deposit Payments Support
- Surcharge Fee Controls
- HPOS & Block Checkout Compatibility

## Features
✔ PCI-Compliant secure gateway  
✔ Google Pay & modern payment flows  
✔ Stripe-style order handling  
✔ Surcharge fee support  
✔ Webhook order sync  
✔ WooCommerce Subscriptions and Deposits  
✔ High-Performance Order Storage (HPOS)  
✔ Classic and Block checkout experience

## Requirements
- WordPress 6.0+  
- WooCommerce 8.0+  
- PHP 8.1+  
- Accept.Blue merchant account

## Installation
1. Upload plugin ZIP to `/wp-content/plugins/`  
2. Activate in WordPress plugins  
3. Configure via WooCommerce → Payments
